❓ Help & User Guide

🧭 Navigation Guide

Use the navigation bar at the top to move between pages:

  • 🏠 Home – Overview of the Language Learning Center with images and course highlights.
  • 🖼️ Gallery – View images of our learning environment and a detailed data table of course schedules.
  • 📝 Booking – Fill out the registration form to enroll in a course.
  • ❓ Help – You are here! Read instructions on how to use the website.
  • 👤 About – Learn about the developer and the project.

📋 How to Use the Booking Form

  1. Fill in your Full Name – this field is required.
  2. Select your Major / Program from the dropdown list.
  3. Choose your Gender by selecting one of the radio buttons.
  4. Check any Hobbies that apply to you (you can select multiple).
  5. Write any Additional Comments or special requests in the text area.
  6. Click "Submit Booking" to send your registration request.
  7. Click "Reset" to clear all fields and start over.

🖼️ Gallery & Data Table Instructions

  • The Gallery displays images of our learning environment. Hover over any image to see a zoom effect.
  • The Data Table shows course schedules, fees, and has links to the booking page.
  • Click "Book Now" inside the table to quickly go to the booking form.
  • External links to Duolingo and Babbel are provided for extra learning resources.
  • The table has alternating row colors and highlights rows when you hover over them.

💡 Tips for Using This Website

  • All pages are responsive – they work on phones, tablets, and computers.
  • The navigation bar stays at the top when you scroll for easy access.
  • Images are optimized for fast loading and have alt text for accessibility.
  • Use the external links to explore more language learning resources.

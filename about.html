📌 Role Summary

As the sole developer of this project, I designed and built a fully responsive website for a fictional Language Learning Center. I implemented all required pages (Home, Gallery, Booking, Help, About) using HTML5 and CSS3.

I focused on accessibility (semantic tags, ARIA roles, alt text), modern layout with Flexbox, and visual enhancements like gradients, shadows, hover effects, and animations. The website aims to provide an intuitive and engaging experience for prospective students.
